# 📚 Manga AI Platform

An AI-powered manga & manhwa reading platform with personalized recommendations and a gamified coin system.

## 🚀 Features
- 🤖 AI-based manga recommendations  
- 📖 Smooth reading experience  
- 🪙 Earn coins by reading chapters  
- 🔓 Unlock latest/premium chapters  
- ❤️ Favorites & reading history  

## 🛠 Tech Stack
- Next.js (Frontend + Backend)
- MongoDB
- OpenAI API
- MangaDex API

## 💡 Concept
Users can read manga, earn coins, and use them to unlock new chapters early while getting smart AI suggestions based on their preferences.
